# 簡略化されたNMタスク

NM評価はすべてのブロックの評価を必要としないかもしれない。これらのブロックでは、NMおよびPQ評価尺度はグレー表示され、テキストに「No Rating Required」と注釈が付けられる。

グレーアウトの結果（「コンテキスト上の結果」とも呼ばれます）は、クエリを理解して評価を知らせるのに役立つ。それらがあなたの評価を決めるのに役に立つと感じると、それをクリックできる。
